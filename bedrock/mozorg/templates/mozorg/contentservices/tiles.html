{{ _('Directory Tiles') }}

{{ _('New users of Firefox with no browsing history will see Mozilla Directory Tiles, a set of default Tiles that may include sponsored content.') }} {{ _('Directory Tiles allow advertisers to reach all of the new Firefox users within a given country, as long as they meet our criteria for new-user content.') }}

{{ _('Suggested Tiles') }}

{{ _('After users browse, they generate their own browsing histories, and Firefox begins to use Tiles to show suggested sites, based on sites that the user has visited frequently and recently.') }} {{ _('Mozilla Suggested Tiles offer sponsored placement in a designated interest category.') }} {{ _('The browser makes these suggestions based on categories of sites that the user has already shown an interest in.') }} {{ _('To protect users’ privacy, Mozilla never shares personally identifiable information, always encrypts data in transmission and promptly discards all data after a week.') }}

{{ _('Suggested Tiles deliver relevant content to the user while respecting their privacy.') }} {{ _('Firefox users remain in complete control of their experience.') }} {{ _('Interest categories are constructed from browsing history in such as not to expose the user’s own history, and all data generated by Tiles is stored on a restricted access server and anonymised within 7 days.') }}

{{ _('Unlike banners and other traditional online ads, tiles appear in an uncluttered grid at the start of a search or site visit, when users are highly receptive to saving time by clicking on a helpful link.') }}

{{ _('Users can always opt out of any Tiles experience, so those who see your advert remain receptive.') }} {{ _('Firefox users can also view the rationale behind every suggestion with a single click — so advertisers are connecting with an engaged audience.') }}
