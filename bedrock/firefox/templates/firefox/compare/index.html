{% trans %} Do you expect a browser - or the company behind it - to respect your privacy? And a little more control over what you share with others wouldn't be bad either? Understandable. {% endtrans %}

{% trans %} We simply expect a minimum level of data protection and privacy from the products we regularly use online. From a browser that we use to browse the web almost every day, the one we use to do our shopping, online banking, social media activities, and lots more, even more so. {% endtrans %}

{% trans %} Therefore, a browser should offer at least a private mode that automatically deletes our history and search history so that other users on the same computer cannot access it. In this regard most browsers and all six of the browsers compared here can score points. {% endtrans %}

{% trans %} But what should be made equally easy for users is to prevent websites and companies from sticking their noses in our data, even in normal browsing mode. Third party tracking cookies are blocked by Firefox and Safari by default and simply put curious advertising trackers outside the door. So here you can decide for yourself what you want to share with others. {% endtrans %}

{% trans %} While all six browsers offer a private mode, an ad filter that blocks certains types of intrusive ads, pop-ups or videos with sound that autoplay is only offered by Firefox, Chrome and Opera. When it comes to cross-site tracking, scripts and blocking autoplay, there are clear differences between the browser features offered. {% endtrans %}

{% trans %} And that’s not just important for privacy: a browser that blocks third-party trackers usually runs much faster too. Trackers run in the background on a number of websites. You can’t see them, but you can feel them slowing down your browser. So it’s worth comparing which browser blocks what. Since the update to version 67, fingerprinting and crypto-mining can also be blocked in Firefox. Especially secret crypto mining can often lead to your computer suddenly being running kinda hot or the battery giving up unusually fast. {% endtrans %}

Almost all six browsers are equal when it comes to tab browsing, bookmark management, auto-completion, proofreading and extensions. {% trans %} The first thing you notice about portability is that not all browsers run on all operating systems. While Firefox, Chrome and Opera work on all major systems and are easy to install, Internet Explorer, Edge and Safari only work on Microsoft and Apple’s own systems. The mobile version of Safari is pre-installed on Apple’s mobile devices, and most Android devices come with a pre-installed browser modified by the manufacturer for the device. Firefox, Chrome, Edge and Opera can easily be installed and used side by side even. {% endtrans %}

{% trans %} Almost all of the browsers compared here allow synchronization between desktop and mobile devices. For this you need an account with all providers with which you can log into the browser on all devices and synchronize bookmarks and settings. {% endtrans %}
